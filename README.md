# 🚀 Basic REST API

A simple and clean REST API built using Node.js and Express.  
This project demonstrates how to create, read, update, and delete (CRUD) data using HTTP methods.

---

## 📌 Features

- RESTful API structure
- CRUD operations
- JSON request & response handling
- Middleware usage
- Clean and beginner-friendly code structure

---

## 🛠️ Tech Stack

- Node.js
- Express.js
